# sgk-telegram-bot

To install dependencies:

```bash
bun install
```

To run for in watch mode:

```bash
npm run dev
```

To build:

```bash
npm run build
```

To run build result:

```bash
npm run start
```

This project was created using `bun init` in bun v1.0.5. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
